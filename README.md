# Backend API Builder Agent

A simple AI-inspired backend planning tool that converts plain English feature requests into a structured backend blueprint.

## Example

Input:

- "Build user auth system"

Output:

- Suggested entities
- API endpoints
- Service classes
- Database tables

## Current MVP

- Accepts a backend feature description
- Generates a structured backend blueprint
- Includes starter logic for auth-related requests
